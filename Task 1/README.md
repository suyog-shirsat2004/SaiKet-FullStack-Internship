# Task 1 — Portfolio Website

**File:** `task1.html`  
**Path on WAMP:** `C:\wamp64\www\Summer Internship\Task 1\task1.html`

## Description
Personal portfolio page for **Suyog Shirsat** — a static frontend project built with HTML, CSS, JavaScript, and Bootstrap 5.3.3.

## Features
- **Hero section** — Profile image, name, tagline ("Full Stack Developer"), CTA buttons
- **Projects grid** — 6 clickable project cards that open a detail modal showing description, tech badges, features list, duration, and role
- **Contact form** — Validates name, email, subject, message; shows green pop-up toast on success
- **Scroll animations** — Sections fade in on scroll
- **Smooth scrolling** — Nav links scroll smoothly to sections
- **Navbar blur effect** on scroll
- **Back-to-top button**
- **Social links** — GitHub → `https://github.com/suyog-shirsat2004`, LinkedIn → `https://www.linkedin.com/in/suyog-shirsat-75a64622a`
