# Task 1 — Static Portfolio Website

**File:** `task1.html`  
**Path on WAMP:** `C:\wamp64\www\Summer Internship\Task-1-Static-Portfolio-Website\task1.html`

## Description
Personal portfolio page for **Suyog Shirsat** — a static frontend project built with HTML, CSS, JavaScript, and Bootstrap 5.3.3. The page is fully responsive and includes interactive features like a project detail modal and form validation.

## Features
- **Hero section** — Profile image, name, tagline ("Full Stack Developer"), CTA buttons with entrance animations
- **Projects grid** — 6 clickable project cards that open a detail modal showing full description, tech stack badges, key features list, duration, and role
- **Contact form** — Validates name, email, subject, message; shows green pop-up toast on success with auto-dismiss
- **Scroll reveal animations** — Sections and cards fade in on scroll via IntersectionObserver
- **Smooth scrolling** — Nav links scroll smoothly to sections
- **Navbar blur effect** on scroll
- **Back-to-top button** — Appears after scrolling down
- **Social links** — GitHub → `https://github.com/suyog-shirsat2004`, LinkedIn → `https://www.linkedin.com/in/suyog-shirsat-75a64622a`

## Requirements
- HTML
- CSS
- JavaScript
- Responsive design
- Visually appealing theme
