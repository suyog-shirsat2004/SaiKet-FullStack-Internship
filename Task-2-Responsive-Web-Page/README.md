# Task 2 — E-Commerce Storefront (ShopVerse)

**File:** `Task2.html`  
**Path on WAMP:** `C:\wamp64\www\Summer Internship\Task 2\Task2.html`

## Description
"ShopVerse" e-commerce landing page built with HTML, CSS, JavaScript, and Bootstrap 5.3.3. Features a dynamic product catalog, shopping cart, checkout order form, dark mode, and newsletter signup.

## Features
- **Hero section** — Animated gradient background with entrance animations
- **Product grid** — 14 products across 4 categories (Electronics, Wearables, Accessories, Footwear) rendered dynamically from JS, grouped section-wise with category headers
- **Product detail modal** — Click any product card to see full info (image, price, badge, category, description)
- **Shopping cart** — Badge counter on cart button; items persist in memory
- **Checkout / Order form** — Click Cart button → opens modal with:
  - Order summary (itemized list + total)
  - Form: Full Name, Email, Phone, Delivery Address, Payment Method (COD / Card / UPI)
  - Full validation on all fields; green success toast with order details
- **Dark mode toggle** — Toggles dark theme across all elements including modals and form
- **Newsletter signup** — Email validation with success/error feedback
- **Scroll reveal animations** — Sections and cards fade in on scroll
- **Navbar blur effect** on scroll
- **Back-to-top button**
